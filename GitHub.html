Curso de Git y GitHub

Repaso de comandos para el manejo del repositorio local

¿Como mostrar cambios en git?

  • $git status
  • $git log
  • $git log --stat
  • $git show
  • $git diff
  • $git branch

¿Como ver los commits y movernos entre ellos en git?

  • $git reset "commit id" (--soft/--hard)
  • $git checkout "commit id"
